About the game

Drugwars is a turn-based strategy computer game in which the player assumes the role of a drug dealer engaged in arbitrage.

The object of the game is to deal the most drugs to pay off the "loan shark" (who charges interest) by the end of the game and make a profit.

Drugwars was originally a DOS program created by John E. Dell in 1984.
